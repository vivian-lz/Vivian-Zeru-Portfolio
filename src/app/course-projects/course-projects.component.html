    (WILL BE COMPLETED) Microcontrollers Final Project (ECE 2218 - Fall 2025)

    Syringe Pump Project (DF 2100 - Fall 2024)
         Our final project in Rapid Prototyping (where DF stands for Digital Fabrication in the course code title) was to create a device that can push a 20 mL and 10 mL syringe utilizing a stepper motor and linear actuator. I worked in a team with myself and three other people to create this device, where our team was called the "Gantry Gang" (given by our professor). We designed our own enclosure using Fusion 360 and 3D printed the components out of PLA. The linear actuator also had 3D printed components made of PLA and ABS (for the motor mount). The device was programmed using an Arduino IDE as it runs on an Arduino Uno microcontroller to create position reset buttons, have a potentiometer to select a flow rate between 0.1 and 5 mL/min, an LCD that displays all of this information, a limit switch that detects how much liquid is in the device, an RGB light that flashes red, yellow, or green to show the status of the device, and a power button for the device. In addition, there is a power supply within the device tiself that allows it to completely run on its own by being plugged into the wall with an outlet plug.
        My main contribution to this project was on the ECE side, as I helped wire up the Arduino together with the breadboard and helped do a lot of the coding of the Arduino Uno Microcontroller.

    Weather Station (Project-Based Programming - HS Senior Year)

    I created a portable weather station in my Project-Based programming class (senior year of high school) using an Arduino Uno and coding on Arduino IDE software that is connected to an LCD and uses RGB lighting and a BME 280 sensor. This device displays the temperature, humidity, pressure, and approximate altitude in the air on the LCD. The device also uses a RGB LED to indicate how hot or cold the weather is through six different color ranges based on the temperature reading (using general known indication where red is the hottest temperature and blue is the coldest with four other colors in between accordingly). Finally, the device is enclosed in a 3D printed case that I created and rendered using OpenSCAD software. I presented each iteration of this project to my class and have tweaked it over the course of a school year before I reached its final product.
